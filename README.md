# She Can Foundation

Single-page static website for She Can Foundation, a women empowerment initiative focused on education, leadership, skill-building, and community engagement.

## What is included

- Responsive landing page
- About, Mission, Impact, and Volunteer sections
- Simple volunteer sign-up form UI
- External hero and about images loaded from Unsplash

## Tech Stack

- HTML5
- CSS3
- Google Fonts

## Run Locally

Open [index.html](index.html) in a browser, or serve the folder with any local HTTP server such as VS Code Live Server.

## GitHub Pages

If you want to publish this project on GitHub Pages, push the repository and set the Pages source to the default branch.
